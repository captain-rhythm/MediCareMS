## Component Breakdown

### JavaScript Functions

#### `showQRCodeModal(invitationId)`
```javascript
// Triggered when admin clicks QR Code button
// Shows loading spinner
// Fetches QR code from server
// Displays modal with QR code
// Handles errors gracefully
```

**Flow:**
```
Button Clicked
  ?
showQRCodeModal(5) called
  ?
Modal shown with spinner
  ?
AJAX: GET /UserManagement/GetQRCodeBase64/5
  ?
Success: Display QR code + link
Error: Show error message
```

#### `copyLinkBtn` Click Handler
```javascript
// Copies invitation link to clipboard
// Shows success feedback
// Auto-resets button after 2 seconds
```

**Sequence:**
```
[Copy] Button Clicked
  ?
Select text from input field
  ?
Copy to clipboard
  ?
Button: "Copy" ? "? Copied!" (green)
  ?
Wait 2 seconds
  ?
Button: "? Copied!" ? "Copy" (blue)
```

#### `downloadQRBtn` Click Handler
```javascript
// Creates temporary link
// Downloads QR code image as PNG
// Names file: invitation-qr-{id}.png
```

**Sequence:**
```
[Download] Button Clicked
  ?
Create blob URL from Base64
  ?
Create temporary <a> element
  ?
Trigger download
  ?
Clean up temporary link
  ?
File saved to Downloads folder
```

## API Response Format

### Success Response
```json
{
  "success": true,
  "qrCode": "iVBORw0KGgoAAAANSUhEUgAAABkA...",
  "invitationLink": "http://localhost:5002/Auth/Register?token=abc123&email=user@example.com"
}
```

### Error Responses
```json
// Invitation not found (404)
{
  "message": "Invitation not found"
}

// Invitation expired (400)
{
  "message": "This invitation has expired"
}

// Invitation already used (400)
{
  "message": "This invitation has already been used"
}

// Server error (500)
{
  "message": "Error generating QR code",
  "error": "Exception details..."
}
```

## Integration Points

### 1. UserManagement Controller
```
GET /UserManagement/GetQRCode/{id}
  ?
ValidationService checks invitation status
  ?
QRCodeService.GenerateQRCodePNG(link)
  ?
Returns PNG file with appropriate headers
```

### 2. User Invitations Table
```
For each invitation:
  ?
If Pending AND Not Used AND Not Expired:
  ?
Show purple "QR Code" button
  ?
onclick ? showQRCodeModal(id)
```

### 3. Modal Component
```
_QRCodeModal.cshtml loaded in layout
  ?
JavaScript event listeners attached
  ?
Modal shown/hidden via Bootstrap
  ?
AJAX calls to controller endpoints
```

## Performance Metrics

### Load Times
```
Button Click           ? 0ms (instant)
Modal Open             ? 50-100ms (CSS animation)
AJAX Request           ? 50-200ms (network)
QR Generation          ? 50-100ms (server-side)
Image Display          ? <50ms (Base64 decode)
Total Time             ? ~200-350ms (user perceivable)
```

### File Sizes
```
QR Code PNG            ? 2-5 KB (depends on URL length)
Base64 Encoded         ? 3-7 KB (same content, different format)
Modal JavaScript       ? 2 KB (inline in partial)
Modal CSS              ? <1 KB (inline in partial)
```
